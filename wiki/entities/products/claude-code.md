# Claude Code

## 定义

Claude Code 是 Anthropic 推出的 **AI 编程助手**，通过命令行界面与开发者交互，帮助编写、修改和调试代码。它是 Claude 模型在软件开发场景下的专业应用。

## 核心特性

### 1. Skill 系统
- 可扩展的插件机制
- 社区贡献的 Skill 生态
- 常用 Skill：
  - **planning-with-files**：基于文件的规划
  - **git**：Git 操作辅助
  - **web-search**：网络搜索
  - **skill-creator**：创建新 Skill

### 2. 多模型支持
- Claude 3.5 Sonnet
- Claude 3 Opus
- Claude 3 Haiku

### 3. MCP (Model Context Protocol)
- 标准化的模型上下文协议
- 支持外部工具集成
- 可连接数据库、API、文件系统等

## 使用场景

- **代码重构**：批量修改代码结构
- **功能开发**：根据需求编写新功能
- **Bug 修复**：分析和修复代码问题
- **代码审查**：自动化代码质量检查
